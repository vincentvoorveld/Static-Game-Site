# Sales Coin Catcher

A very simple browser game built as a 100% static site using HTML, CSS, and vanilla JavaScript. 
No backend, no external libraries, no frameworks!

## Game Concept
You are a salesperson. Catch the good items (3C coins, Quotes, Banknotes) to earn points. Avoid the bad items (Meetings, Spam, No-shows, Admin work) or you will lose a life!

## How to play locally
1. Clone or download this repository.
2. Open `index.html` directly in any modern web browser.
3. Use the Left/Right arrow keys (or on-screen buttons / touch dragging on mobile) to move.

## How to deploy to GitHub Pages
1. Push this repository to GitHub.
2. Go to your repository **Settings**.
3. Navigate to **Pages** in the left sidebar.
4. Under "Build and deployment", select **Deploy from a branch**.
5. Select the `main` (or `master`) branch and the `/ (root)` folder.
6. Click **Save**. Your game will be available at your GitHub Pages URL shortly!
